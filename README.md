## ğŸ§  What It Does

The system uses a dog clutch with a triple-gear mesh design to select between forward and reverse spur gears. A step-down gear
on the input shaft drives a secondary shaft with a bevel gear, which engages two bevel gears connected to spur gears on the output shaft. For braking, fins on the dog clutch engage a block on the base when shifted to neutral, effectively stopping the system. This design ensures efficient power transfer, directional control, and reliable braking.
